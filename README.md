## 为什么使用Karma
- 真实浏览器
- 多浏览器
- 开发过程中本地运行
- 持续集成服务器中运行
- 每次保存自动运行
- 终端运行
- 生成测试报告
- 模块化，es6支持

## 安装
```bash
$ npm install karma --save-dev

// 可以全局使用karma命令
$ sudo npm install -g karma-cli
```

## 配置
```
$ karma init
```
## 运行
```
$ karma start my.conf.js
```
## 查看start可用参数
```
$ karma start --help
```
